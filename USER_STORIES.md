# User Stories

1. As a new user, I want to register with username, email, and password so I can create an account.
2. As a returning user, I want to log in with email and password so I can access the app.
3. As a user, I want to see a home screen with a logo and my habit list so I can track my progress.
4. As a user, I want to open a detail screen so I can view full habit information.
5. As a user, I want my habit data saved locally so it remains after closing the app.
6. As a user, I want motivational content fetched from an API so I stay encouraged.
7. As a user, I want a settings menu so I can manage app options.
8. As a user, I want to switch themes so I can use light or dark mode.
9. As a user, I want notification reminders so I can remember my habits.
